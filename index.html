🎨 Bring this into Canva

  1. Tap Save — downloads a .webm file.
  2. Save the file to your phone's gallery (most phones auto-convert to .mp4).
  3. Open Canva → Create a design → blank video.
  4. Tap Uploads → Upload media → select your saved video.
  5. Drag onto the timeline, then add text, music, transitions.
The top-left of each scene is left blank so you can overlay your own text in Canva.
